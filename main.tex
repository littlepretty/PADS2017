\begin{abstract}
Software defined network abstracts the forwarding logic of the distributed networking
system in order to make programming network devices to be centralized, simple and efficient.
In this work we propose to leverage this paradigm in emulating large scale SDN network.
Given a snapshot of the SDN powered network,
we can cluster any number of OpenFlow devices into one virtualized switches
that preserve packet-level fidelity.
Our long term goal is to abstract the clustered portion of the SDN network so that
emulating one OpenFlow switch is equivalent to a great number of interconnected
OpenFlow switches in terms of both forwarding logics and network performance metrics.
In this paper we focus on the logical part first and design systematic method and various
algorithms to efficiently aggregate the distributed forwarding rules installed on a group
of OpenFlow switches.
Preliminary experiment results show that the rules on the abstracted ``big" switch
can be compactly expressed without loss of forwarding correctness.
Thus a large amount of emulating resources (rules and switches in the network) can be reduced via just the first logic abstraction stage.
\end{abstract}
